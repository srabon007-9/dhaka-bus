🚀 QUICK STEPS:

Step 1: CREATE GITHUB REPO
  • Go to github.com
  • Create new repo "dhaka-bus"
  • Choose: Public, Skip initialization

Step 2: PUSH YOUR CODE
  cd dhaka-bus
  git init
  git remote add origin https://github.com/YOUR-USERNAME/dhaka-bus.git
  git add .
  git commit -m "Initial commit"
  git push -u origin main

Step 3: INVITE FRIENDS
  • Go to GitHub repo Settings → Collaborators
  • Click "Add people"
  • Enter friend's GitHub username
  • Choose role: "Write" (for team members)
  • Send invitation

Step 4: SHARE FRIEND_GUIDE.md
  • Copy content from FRIEND_GUIDE.md
  • Send to your friends
  • They're ready to contribute!
